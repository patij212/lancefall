# Development notes (archive)

Internal planning, design-iteration, and per-feature **handoff** notes kept from
LANCEFALL's development. They live here to keep the repo root and `docs/` focused
on the player- and contributor-facing documentation.

These are point-in-time working notes, **not** maintained documentation. For the
current state of the game see the top-level [`README.md`](../../README.md); for the
design record see [`DESIGN.md`](../../DESIGN.md) and its per-iteration log
[`DEVLOG.md`](../../DEVLOG.md).

Contents:

- `*_PLAN.md`, `NEXT_SESSION*.md`, `HANDOFF.md` — session planning + handoffs
- `AUDIO_*.md` — the audio-system design / plan log
- `HANDOFF-*.md` — per-feature implementation handoffs (parry, enemies, bosses,
  modes, onboarding, player accounts, the codebreaker console, …)
